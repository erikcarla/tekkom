How to use:

1. Click next to proceed to the next step.
2. Select Type Checking to change the data types of variables adapted to the type of data in the compiler.
3. Fill the token.
4. Fill Table Manager with variable and select the desired data type.
5. Fill the Statement.
6. Hover the part of tree to see the detail.

NB: Target user = Mahasiswa IT, Expert User.

  • Semantic is included in the analysis process. In the analysis 3 processes occur. They are lexical analysis, syntactic analysis and semantic analysis. In the lexical analysis the characters that make up the source program is read from left to right and grouped in what is called a token (scanning). Then after scanning, character or token obtained on a lexical analysis are compiled and classified in a particular hierarchy as a whole has a certain meaning. After all that processes, final structure that has been obtained will be checked and examined its suitability with existing program components, known as semantic analysis.
